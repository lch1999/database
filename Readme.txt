在sjtu的校园网内可以直接访问http://lilab.life.sjtu.edu.cn:8080/nhlpd/ribosomal/index.php查看数据库。
若不在校园网内，可在本地浏览。方法如下：解压phpstudy，运行phpstudy.exe。在浏览器中输入localhost/ribosomal/即可运行。
数据库原文件：ribosomal，drug_resistant_strain_ribosome。
drug_resistant_strain_ribosome为mysql的data。如本地配置好运行环境可只下载两个原文件。
在phpstudy中ribosomal位于PHPstudy\PHPTutorial\WWW。
drug_resistant_strain_ribosome位于PHPstudy\PHPTutorial\MySQL\data
php版本为3.29
